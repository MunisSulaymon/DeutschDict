About DeutschDict

DeutschDict is a complete, production-ready German dictionary web app designed for language learners. It offers definitions in 18 languages, powered by a suite of AI agent patterns to make learning intuitive, engaging, and effective.

AI Papers & Patterns Powering DeutschDict

  • ReAct (Reasoning + Acting): Search thinking steps visibly show reasoning in real time (Detecting input → Fetching definitions → Translating).
  • Reflexion: The quiz memory stores wrong answers and auto-generates reflection notes to build spaced repetition.
  • RAG (Retrieval-Augmented Generation): The dictionary strictly follows a retrieval pipeline to fetch definitions from external knowledge before translating, preventing hallucinations.
  • Chain-of-Thought: Word of the Day tips break down meaning into actionable context, showing why a word is important.
  • Agent Memory: A comprehensive learning profile system using In-context, External, Semantic, and Episodic memory stored directly on your device.
